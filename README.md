# GamepadNav

Xbox controller → mouse/keyboard input for Windows. Enter PINs, navigate the desktop, and control your PC from the couch — including the login screen.

## Features

- **Mouse emulation** — left stick moves cursor, right stick scrolls
- **Button mappings** — RB/LB for clicks, d-pad for arrows, triggers for Shift/Ctrl
- **Virtual keyboard** — QWERTY + numpad overlay, controller-navigable
- **Login screen support** — works on Windows lock/login screen for PIN entry
- **Auto-disable** — detects games and stops translating input
- **Manual toggle** — click both thumbsticks (L3+R3) to enable/disable

## Requirements

- Windows 11
- Xbox controller (wireless adapter or Bluetooth)
- .NET 8 Runtime
